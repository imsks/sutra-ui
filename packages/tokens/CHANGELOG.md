# @sutra/tokens

## 0.0.1

### Patch Changes

- Initial release of Sutra.

  - `@sutra/tokens`: color, spacing, typography, radii, and shadow tokens as TypeScript constants, CSS variables (`:root` + `.dark`), and a Tailwind v4 preset. Themeable warm brand accent; Rozha One + Mukta typography (Devanagari + Latin).
  - `@sutra/ui`: ten accessible, TypeScript-first React primitives — Button, Card, Badge, Avatar, Skeleton, Field, Input, Textarea, Select (styled native), Modal, and a Toast system — plus a curated icon set at `@sutra/ui/icons`. Dark mode built in, zero runtime CSS-in-JS.
